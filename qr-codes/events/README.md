# QR Codes - events

QR codes for specific event registration pages and event check-in

## How to add QR codes here
1. Generate your QR code (recommended: qr-code-generator.com or Canva)
2. Export as PNG at minimum 500x500px
3. Name the file descriptively: e.g. `twizted-website-qr-2026.png`
4. Drop the file in this folder

## Naming convention
`[purpose]-[destination]-qr-[year].png`

Example: `facebook-page-qr-2026.png`
