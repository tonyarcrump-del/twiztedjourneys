# Twizted Journeys - QR Code Library

## Folder Structure
- `/website` - Main site QR codes
- `/events` - Event-specific QR codes
- `/donations` - Donation page QR codes
- `/facebook` - Facebook page QR codes
- `/merch` - Merch store QR codes
- `/memorials` - Memorial page QR codes
- `/future-campaigns` - Reserved for upcoming campaigns

## Generation Tools
- [QR Code Generator](https://www.qr-code-generator.com/)
- [Canva QR Tool](https://www.canva.com/qr-code-generator/)
- [GoQR](https://goqr.me/)

## Standards
- Minimum size: 500x500px PNG
- Include logo center embed when possible (Twizted semicolon mark)
- Always test scan before printing
- Keep high-res version + print-ready version
